# XIV Practice Tools

Practice FFXIV raid mechanics interactively. Pick an encounter, mechanic, strategy, and your role — then click where you'd stand on the arena.

The tool tells you if you were right, and shows you the correct spot if you weren't.

## Supported content

### M11S — Tyrant

- Arena Split → Outer Waymarks

### M12S P2 — Lindwurm II

- Replication 2 → Banana Codex

## Hints

Each phase has an optional **Show Hints** button. When enabled, your correct position is shown as a faint overlay on the arena. Turn it off once you no longer need it.
